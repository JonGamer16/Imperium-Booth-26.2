# **Predicates**

summit.booth:\<namespace\>/in\_booth 

- Passes if the current location is within any of the bounding boxes of the booth specified by the namespace defined in the [Booth Definition](?tab=t.5lmhbhoqu627).

summit.booth:\<namespace\>/in\_bounding\_box/\<bounding\_box\_name\>

- Where \<bounding\_box\_name\> is the name of the bounding box in the [Booth Definition](?tab=t.5lmhbhoqu627)  
- This will pass if the current location is in this specific bounding box
