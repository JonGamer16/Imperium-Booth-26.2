# **Function Tags**	

\#summit:api/reset\_player 

- In your booth you might be modifying players in various ways. This function tag is called to reset a players’ state. If you modify the player with attribute modifiers or anything similar for your booth, please add a function to this tag.  
- Note, this does **not** reset a player’s progress on persistent objectives such as stickers.
