# **Summit Assets** 

## Reserved Block States ![][image1]

There are certain block states that are reserved and you should not build with. The items for these blocks are marked with a Blue ![][image2] overlay, and further block states that are reserved have a similar texture as their model.

Notably, all unwaxed copper blocks are reserved or overwritten.

## Overwritten Block States

As a rule, booth vendors are not allowed to overwrite vanilla block states.   
However there are a number of overwritten block states that you can use \- far too many to actually list out here, but we’ve ensured that there is at least one vanilla block state still accessible for you to build with. 

The most notable overwritten block states are: 

- **Cherry Leaves**   
  - For non-persistent cherry leaves, each distance will correspond to a different leaf color.  
- **Sugar Cane**  
  - Age 1 \= White Mushroom  
  - Age 2 \= Tall Wheat  
  - Age 13 \= Lower Shorter Grass  
  - Age 14 \= Clovers  
  - Age 15 \= Shorter Grass  
- **Mangrove Propagule**  
  - Used for various custom flowers, accessible from the quick actions menu (G) with the BSL datapack installed.  
- …

## Barrier Block States

Several unwaxed oxidized copper blocks have been turned into barrier-esque blocks for your use. These are marked with a smaller barrier symbol in the corner of the item icon.

## Use of Custom Entity Models

**Ask for permission to use any of them.** We know people like the summit models, but we want to ensure that the usage of these makes sense for the areas they’re in to maintain the visual style that makes Summit Summit.

## Pixel Twemoji

[Pixel Twemoji 9x](https://modrinth.com/resourcepack/pixel-twemoji-9x) will be used on the production Summit server, as such it is bundled with the BSL resource pack.
